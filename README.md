# Nostr App

Decentralized blogging app using Nostr protocol.

I built this to explore the [Nostr protocol](https://github.com/nostr-protocol/nostr). Information about [NIPs found here](https://github.com/nostr-protocol/nips).

Relies on the [nostr-tools](https://github.com/nbd-wtf/nostr-tools) package.

## Getting Started

First, install packages:

```bash
yarn
```

run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
